My SnapChat

In this project you can send text messages, pictures or videos to your friends. Then you can choose the duration they can watch the message. This project has been done following agile method Scrum in a team of 5 developers.We used Cordova to build the website as a mobile app, MEAN stack is use and tested with Protractor. DYF Scrum

This is a project management website for the SCRUM method. You can create a project, add user stories, assign it to a Sprint, create tasks, link it with an user story and assign the task to an user. During the project you can follow the evolution with the kanban
